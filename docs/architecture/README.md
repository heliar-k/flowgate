# FlowGate Architecture

This folder contains C4 architecture documentation for the FlowGate project.

- `c4-context.md`: Level 1 system context.
- `c4-containers.md`: Level 2 container view.
- `c4-components-cli.md`: Level 3 component view for the FlowGate CLI codebase.
- `c4-deployment.md`: Level 4 deployment view for local development runtime.
- `c4-dynamic-bootstrap-download.md`: Dynamic flow for runtime bootstrap artifact setup.
- `c4-dynamic-profile-switch.md`: Dynamic flow for profile switch and conditional service restart.
