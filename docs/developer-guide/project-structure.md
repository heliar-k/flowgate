# Project Structure

High-level layout:

- `src/flowgate/`: main package (CLI, config, process, profile, auth, bootstrap)
- `config/examples/`: example configs
- `docs/`: documentation (user guide, architecture, developer notes)
- `tests/`: unit + integration tests
- `scripts/`: helper scripts (`xgate`, `xtest`, smoke/security checks)

Architecture diagrams: `docs/architecture/README.md`.
